# GupShupGo

A production-ready Flutter app with HD video calling, real-time messaging and status updates. Open source, MIT licensed.

## Features

- HD Video Calling (Agora RTC)
- Voice Calls with noise suppression
- Real-time Messaging via Firestore
- Status Updates with 24-hour auto-expiry
- Phone Authentication (Firebase)
- Push Notifications
- Call Logs & Presence System

## Tech Stack

- **Frontend**: Flutter 3.5, Dart
- **Backend**: Firebase (Firestore, Auth, Cloud Messaging)
- **Video/Voice**: Agora RTC Engine
- **Landing Page**: React, Vite, TypeScript, Tailwind CSS, shadcn/ui

## Getting Started

```sh
# Clone the repository
git clone https://github.com/vansh-121/GupShupGo.git

# Navigate to the project directory
cd GupShupGo

# Install dependencies
npm i

# Start the development server
npm run dev
```
